# PCA from Scratch

Starter repository for the **Linear Algebra for ML** capstone on [DutchEngineer](https://dutchengineer.org).

## What you're building

A PCA implementation built on SVD — no `sklearn.decomposition.PCA` — validated against scikit-learn on a real dataset. The constraint forces you to work through the linear algebra directly rather than relying on an abstraction to hide it.

## Getting started

1. Fork this repository
2. Clone your fork: `git clone https://github.com/<your-username>/pca-from-scratch-starter`
3. Follow the capstone instructions in your DutchEngineer dashboard
